1. Two Sum

Problem

Given an array of integers, return indices of the two numbers such that they add up to a specific target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

Example 1:

Given nums = [2, 7, 11, 15], target = 9,
Because nums[0] + nums[1] = 2 + 7 = 9,
return [0, 1].

Example 2:

Given nums = [3,2,4], target = 6
Because nums[1] + nums[2] = 2 + 4 = 6,
return [1, 2].

Solution:

Brute Force:

Loop through all possible pairs in the list.

Complexity Analysis:

  • Time complexity : O(n2)O(n^2)O(n2). For each element, we try to find its complement by looping through the rest of array which takes O(n)O(n)O(n) time. Therefore, the time complexity is O(n2)O(n^2)O(n2).
  • Space complexity: O(1)O(1)O(1).
def twoSum(self, nums, target):
        for i in range(len(nums)):
	        for j in range(i+1,len(nums)):
		        if nums[i] + nums[j] == target:
			        return [i,j]

One Pass:

While iterating and inserting elements into the hash table, we also look back to check if current element’s complement already exists in the table. If it exists, we have found a solution and return immediately.

Complexity Analysis:

  • Time complexity : O(n)O(n)O(n). We only need to iterate through the list once, and each look up in the dictionary takes O(1)O(1)O(1).
  • Space complexity : O(n)O(n)O(n). Depending on the number of items stored in the hash table, which is at most nnn.
def twoSum(self, nums, target):
        d = {}
        for i, n in enumerate(nums):
            m = target - n
            if m in d:
                return [d[m], i]
            else:
                d[n] = i
